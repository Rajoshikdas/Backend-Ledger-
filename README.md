# Backend-Ledger-
Bank Transaction System – Full Stack Application 
Node.js · Express.js · MySQL · React.js · RESTful APIs · Git
•	Built a secure full stack banking application using Node.js and Express.js backend with NoSQL database, designing schemas for accounts and transactions.
•	Designed and integrated RESTful APIs for deposits, withdrawals, fund transfers, and transaction history — implementing complete request-response lifecycle with validation and error responses.
•	Developed React.js dashboard with real-time balance updates and transaction logs, applying component-based frontend development and state management with hooks.
•	Maintained codebase using Git with feature-branch workflow, practising version control best practices throughout the development lifecycle.
